## 結構脈絡（出處見專案 report.html / data/coreweave.js）

- 債務 15 個月 3 倍：$8.0B（2024 底）→$21.6B（2025 底）→$25.1B（Q1'26）+ Q1 後 ~$9.9B
- 含表外總索賠 ~$75-90B vs 調整後 EBITDA $3.1B ≈ 24×；表外未開始租約 $40.7B+（至 2029 起租）
- 母公司擔保：除 DDTL 4.0（bad-boy）外全部無條件擔保——SPV 違約直接回傷母公司
- DDTL 4.0 advance rate 穩定後可達 ~102% LTC；受限現金瀑布先償債後放款
- Nvidia 三重角色：股東 + 供應商 + $6.3B 回購背書（2032/4 前，**動用＝需求疲弱訊號**，留意揭露）
- GPU 租金市價兩年跌 60-75%——債靠現金流還，不靠「還能用的晶片」
- 對手方風險集中且互鎖：MSFT（FY25 營收 67%）、Meta（$14.2B+$21B）、OpenAI（$22.4B、
  正在砍承諾）、Cohere、Anthropic——客戶端惡化看 openai/anthropic/cohere skill
- 參議院 FSOC 信函點名「第一張倒下的骨牌」；首檔公開交易 GPU 債＝市價放大器
